# 🚨🚨 Girl Safety Alert System 🚨🚨

A full-stack web application designed to enhance personal safety by enabling users to send real-time SOS alerts with live location to emergency contacts.

---

## 🌟 Features

- 🔐 Secure Login & Registration (JWT Authentication)
- 🚨 One-click SOS Alert system
- 📍 Real-time Location Sharing (Google Maps link)
- 📧 Email Notifications to Emergency Contacts
- 👥 Add & Manage Emergency Contacts
- 🎯 Fast response system for emergencies

---

## 🛠️ Tech Stack

- Frontend: React (Vite)
- Backend: Spring Boot
- Database: MySQL
- Authentication: JWT
- APIs: RESTful APIs
